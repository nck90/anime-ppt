백준허브 설치하기

크롬 확장 프로그램으로 간편하게 설치하고 GitHub와 연동하기

1 크롬 웹 스토어 방문하기

Chrome 웹 스토어에서 "BaekjoonHub" 또는 "백준허브"를 검색하세요.

크롬 웹 스토어 바로가기

2 확장 프로그램 설치하기

백준허브 확장 프로그램 페이지에서 "Chrome에 추가" 버튼을 클릭하여 설치합니다.

설치 후 브라우저 상단의 확장 프로그램 아이콘이 보입니다. 아이콘을 고정하면 더 편리하게 사용할 수 있습니다.

3 GitHub 계정 연동하기

확장 프로그램 아이콘을 클릭하고 "Authenticate" 버튼을 눌러 GitHub 계정을 연동합니다.

GitHub에 로그인하여 인증을 완료합니다.
"Authorize BaekjoonHub" 버튼을 클릭하여 권한을 부여합니다.

4 리포지토리 설정하기

GitHub 인증 후 자동으로 리포지토리를 생성하거나 기존 리포지토리를 선택할 수 있습니다.

백준과 프로그래머스 문제는 각각 다른 리포지토리에 자동으로 저장됩니다. 필요에 따라 리포지토리 설정을 변경할 수 있습니다.
