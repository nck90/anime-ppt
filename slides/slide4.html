Git 주요 명령어

일상적인 개발 워크플로우에서 자주 사용되는 Git 명령어

git init

새로운 Git 저장소 초기화

.git 하위 디렉토리 생성

git clone

원격 저장소 복제

원격 저장소의 전체 복사본 다운로드

git status

작업 디렉토리 상태 확인

추적/변경/스테이징된 파일 목록 표시

git add

변경사항을 스테이징 영역에 추가

개별 파일 또는 전체(git add .) 추가

git commit

변경사항을 저장소에 기록

메시지 포함(git commit -m "메시지")

git branch

브랜치 목록 확인 및 관리

새 브랜치 생성(git branch [이름])

git checkout

브랜치 전환

새 브랜치 생성 및 전환(git checkout -b [이름])

git push/pull

원격 저장소와 동기화

push: 로컬→원격, pull: 원격→로컬
