백준허브란?

백준, 프로그래머스 등의 코딩 테스트 플랫폼에서 해결한 문제를 자동으로 GitHub에 커밋하는 크롬 확장 프로그램입니다.

자동 코드 저장

문제를 해결하면 자동으로 GitHub 리포지토리에 코드가 커밋되어, 별도의 작업 없이 코딩 기록을 관리할 수 있습니다.

포트폴리오 구축

풀이한 문제들이 체계적으로 GitHub에 기록되어 개발자로서의 문제 해결 능력을 보여주는 포트폴리오가 됩니다.

다중 플랫폼 지원

백준, 프로그래머스 등 여러 코딩 테스트 플랫폼을 지원하여 다양한 문제 풀이를 한 곳에 모을 수 있습니다.

코딩 학습 추적

문제 해결 이력을 시각적으로 확인하고 추적하여 학습 진도와 성장을 한눈에 파악할 수 있습니다.
